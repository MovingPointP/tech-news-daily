### [Node.js 26.0.0 (Now with Temporal)](https://news.ycombinator.com/item?id=48212242)
Node.js 26.0.0 が2026年5月5日にリリースされました。
最大の特徴は、Temporal API がデフォルトで有効になったことで、JavaScript の古い Date オブジェクトを置き換えます。
Temporal は時間帯、カレンダー、期間、インスタントをより豊富に扱うことができ、JavaScript の日付処理における長年の問題を解決します。
同時に V8 JavaScript エンジンが 14.6 にアップデートされ、Map.prototype.getOrInsert() などの新機能が追加されました。
Node.js 26 は Current リリースとして登場し、10月2026年に LTS 状態に移行する予定です。

## Hacker News

### [Biff is a command line datetime Swiss army knife](https://news.ycombinator.com/item?id=48303881)
Biffはdatetime演算、パース、フォーマットを統合したRustで実装されたコマンドラインツールである。
タイムゾーン変換、期間の加算、特定日付からの経過時間計算など多機能を備えている。
RFC3339フォーマット、ロケール対応フォーマッティング、日付シーケンス生成など豊富な機能がある。
複数のOSで静的バイナリが提供され、cargoでのインストールも可能である。
Jiffのdatetime処理とICU4Xのローカライズ機能を活用した設計となっており、ライセンスはMITまたはUNLICENSEである。

### [Uber blows through its AI budget in 1 quarter](https://news.ycombinator.com/item?id=48287025)
Uberは2026年の年間AI予算をわずか4ヶ月で使い尽くし、2つの主要なAIコーディングツールであるClaude CodeとCursorへの投資の効果を疑問視している。
エンジニアあたりの月額API費用は500ドルから2000ドルの範囲で、Uberのエンジニア95%が月間でAIツールを使用している。
提供されるコードの70%がAI由来であるにもかかわらず、COO Andrew Macdonaldは「消費者向けサービスのイノベーションとの直接的な関連性を見出せていない」と述べている。
ROIの明確な関連付けがないため、今後はAIトークン消費コストを直接エンジニア採用コストと比較する必要があるという方針に転換している。
この事例は、高い採用率と実際の生産性向上が必ずしも一致しないことを示唆している。

### [Cursor 3.3 vs Claude Code — PRレビュー設計の判断基準](https://zenn.dev/miyan/articles/ai-code-cursor-claude-pr-review-2026)
Cursor 3.3（2026年5月リリース）はBugbotとSecurity Reviewer Betaを搭載し、Claude Code v2.1.111の/ultrareview関数と異なるPRレビュー階層を提供している。
Cursor TeamsとEnterprise向けのSecurity Reviewerはベータ段階であり、個別開発者はBugbot、/ultrareview、CodeRabbit、Greptileなどの代替手段を検討する必要がある。
Cursor Teams $40/ユーザー/月に対し、/ultrareviewは$5-$20/PR単位の価格設定となっており、月4PR以上のチーム規模で経済性が逆転する。
Bugbotはコード品質や論理エラー、セキュリティ脆弱性を検出し、Default設定で0.7バグ/レビュー、High設定で0.95バグ/レビューを検出する。
これらのツールはGoohartの法則を避けるため、公開ベンチマークのみに依存せず、実際のワークフロー要件に基づいた選択が重要である。

2026年のAIコーディングツール市場は、単一ツール選択からマルチエージェント戦略への転換期を迎えており、開発組織はこれに対応する必要がある。
